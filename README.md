# AdPal Home

This is a code bundle for AdPal Home.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## About AdPal

AdPal is a platform that brings monetizable "green" display screen to users. AdPal developed built-in verifiable display technology in all screens delivered, turning each digital display into a digital billboard. Users browse available ad campaigns from brands, commit their screens to display ads for a set period (typically 4–8 hours), and earn cryptocurrency tokens in return. The longer display, the more earnings. Users can also pick up extra tokens by completing quick bonus tasks throughout the day. When not earning, the screen isn't wasted — users can cast free content from a public gallery or self-uploads to keep it useful and personalized. It's essentially passive income from a screen that's just sitting on desk, shelf, or wall. The main hardware product is a eink-screen attached to iPhone case/shell, to powerbanks, digital photo frames, etc.
